# 金蝶云星空 - 会计凭证管理模块 - 列表过滤方案

## 一、凭证列表过滤方案设计

### 1.1 标准过滤方案

#### 方案一：全部凭证
- **方案名称**：全部凭证
- **方案类型**：系统预设
- **过滤条件**：
  - 无限制条件
- **排序规则**：
  - 按凭证日期降序
  - 按凭证号降序

#### 方案二：本期凭证
- **方案名称**：本期凭证
- **方案类型**：系统预设
- **过滤条件**：
  - 会计年度 = 当前年度
  - 会计期间 = 当前期间
- **排序规则**：
  - 按凭证日期升序
  - 按凭证号升序

#### 方案三：未审核凭证
- **方案名称**：未审核凭证
- **方案类型**：系统预设
- **过滤条件**：
  - 单据状态 = 制单
- **排序规则**：
  - 按凭证日期升序

#### 方案四：已审核未记账
- **方案名称**：已审核未记账
- **方案类型**：系统预设
- **过滤条件**：
  - 单据状态 = 已审核
- **排序规则**：
  - 按凭证日期升序

#### 方案五：已记账凭证
- **方案名称**：已记账凭证
- **方案类型**：系统预设
- **过滤条件**：
  - 单据状态 = 已记账
- **排序规则**：
  - 按凭证日期降序

#### 方案六：我的凭证
- **方案名称**：我的凭证
- **方案类型**：系统预设
- **过滤条件**：
  - 创建人 = 当前用户
- **排序规则**：
  - 按创建时间降序

### 1.2 自定义过滤字段

#### 常用过滤字段
| 字段标识 | 字段名称 | 字段类型 | 过滤方式 |

| FDOCUMENTSTATUS | 单据状态 | 枚举 | 多选 |
| FVOUCHERTYPEID | 凭证类型 | F7选择 | 多选 |
| FACCOUNTID | 科目 | F7选择 | 多选 |
| FPERIODID | 会计期间 | F7选择 | 多选 |
| FCREATORID | 创建人 | F7选择 | 多选 |
| FAUDITORID | 审核人 | F7选择 | 多选 |
| FPOSTERID | 记账人 | F7选择 | 多选 |
| FATTACHMENTCOUNT | 附件数 | 整数 | 等于、大于、小于、区间 |
