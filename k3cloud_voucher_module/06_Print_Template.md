## 二、标准A4记账凭证模板

### 2.1 页面布局
- **纸张规格**：A4 210mm × 297mm
- **页边距**：上20mm、下20mm、左25mm、右20mm
- **方向**：纵向
- **字体**：微软雅黑，9pt-12pt

### 2.2 页眉区域

#### 公司名称
- **位置**：页面顶部居中
- **字体**：微软雅黑，16pt，加粗
- **内容**：{组织名称}
- **示例**：银河星辰科技有限公司

#### 凭证标题
- **位置**：公司名称下方居中
- **字体**：微软雅黑，18pt，加粗
- **内容**：记 账 凭 证

#### 凭证信息栏
- **位置**：标题下方，左右两栏
- **左侧内容**：
  - 凭证字：{凭证类型}
  - 凭证号：{凭证号}
  - 附单据数：{附件数} 张
- **右侧内容**：
  - 日期：{凭证日期}
  - 会计期间：{年度}年{期间}期
- **字体**：微软雅黑，10pt
- **对齐**：左对齐/右对齐

### 2.3 凭证明细表格

#### 表头
| 列号 | 列名 | 宽度 | 对齐 | 备注 |
|-----|------|------|------|------|
| 1 | 摘要 | 180px | 左 | |
| 2 | 科目编码 | 80px | 左 | |
| 3 | 科目名称 | 140px | 左 | |
| 4 | 借方金额 | 120px | 右 | 分位线 |
| 5 | 贷方金额 | 120px | 右 | 分位线 |

#### 借方/贷方金额格式
- 显示格式：千分位分隔，2位小数
- 分位线：显示"千百十万千百十元角分"
- 示例：
```
      借方金额              贷方金额
千百十万千百十元角分  千百十万千百十元角分
  1  2  3  4  5  6    0  0  0  0  0  0
```

#### 表体行数
- 默认显示：10行
- 不足补空行
- 超过分页显示

### 2.4 合计行
- **位置**：明细表格下方
- **内容**：
  - 摘要列：合计
  - 借方金额：{借方合计}
  - 贷方金额：{贷方合计}
- **样式**：
  - 加粗显示
  - 下划线
  - 金额大写在右侧

### 2.5 金额大写区域
- **位置**：合计行右侧
- **内容**：人民币{大写金额}
- **字体**：微软雅黑，10pt，加粗
- **示例**：人民币壹万贰仟叁佰肆拾伍元陆角柒分整

### 2.6 制单人签章区域
- **位置**：页面底部
- **布局**：从左到右排列
- **签章项**：
  1. 制单：{制单人}
  2. 审核：{审核人}
  3. 记账：{记账人}
  4. 出纳：{出纳人}
  5. 主管：{主管人}
- **样式**：每个签章项下方留空用于盖章

### 2.7 页脚
- **位置**：页面底部居中
- **内容**：第{当前页}页 / 共{总页数}页

## 七、打印模板配置方法

### 7.1 BOS模板配置步骤

#### 1. 进入BOS设计器
```
BOS IDE → 业务对象 → 会计凭证 → 打印模板
```

#### 2. 新建打印模板
- 点击"新建模板"
- 选择模板类型（简单报表/复杂报表）
- 选择纸张规格
- 设置页面布局

#### 3. 配置数据源
- 主表数据源：`T_VOUCHER`
- 明细表数据源：`T_VOUCHERENTRY`
- 关联关系：`FID = FPARENTID`

#### 4. 拖拽数据字段
- 从数据源面板拖拽字段到设计区域
- 设置字段格式、对齐方式
- 配置表达式字段

#### 5. 配置打印参数

```xml
<!-- 打印参数配置示例 -->
<PrintConfig>
  <PageSetup>
    <PaperSize>A4</PaperSize>
    <Orientation>Portrait</Orientation>
    <Margins>
      <Top>20</Top>
      <Bottom>20</Bottom>
      <Left>25</Left>
      <Right>20</Right>
    </Margins>
  </PageSetup>
  <DataFields>
    <Field Name="FBILLNO" Caption="凭证号" />
    <Field Name="FDATE" Caption="凭证日期" Format="yyyy-MM-dd" />
    <Field Name="FDEBITAMOUNT" Caption="借方金额" Format="#,##0.00" />
  </DataFields>
</PrintConfig>
```

### 7.2 打印插件配置

#### C# 打印事件插件
```csharp
using Kingdee.BOS.Core.DynamicForm.PlugIn;
using Kingdee.BOS.Core.DynamicForm.PlugIn.Args;
using Kingdee.BOS.Core.Print;

public class VoucherPrintPlugin : AbstractPrintPlugIn
{
    public override void OnPreparePrint(PreparePrintEventArgs e)
    {
        base.OnPreparePrint(e);
        
        // 获取打印参数
        var billNo = this.Model.GetValue("FBILLNO").ToString();
        
        // 设置打印份数
        e.PrintParameter.Copies = 1;
        
        // 设置是否双面打印
        e.PrintParameter.Duplex = Duplex.Default;
    }
    
    public override void OnBeforePrint(BeforePrintEventArgs e)
    {
        base.OnBeforePrint(e);
        
        // 打印前校验
        var status = this.Model.GetValue("FDOCUMENTSTATUS").ToString();
        if (status != "C" && status != "B")
        {
            e.Cancel = true;
            e.CancelMessage = "只有已审核或已记账的凭证才能打印！";
        }
    }
    
    public override void OnAfterPrint(AfterPrintEventArgs e)
    {
        base.OnAfterPrint(e);
        
        // 记录打印日志
        this.WriteLog("凭证打印成功", "");
    }
}
```
